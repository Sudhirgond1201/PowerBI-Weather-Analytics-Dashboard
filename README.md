# 🌦️ Weather Analytics Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Weather Analytics Dashboard** developed using **Microsoft Power BI**. The dashboard provides comprehensive insights into weather conditions across multiple cities by analyzing **temperature, humidity, pressure, wind speed, wind direction, and weather descriptions**.

The objective of this project is to transform raw weather data into meaningful business insights through interactive visualizations and dynamic KPIs.

---

## 🚀 Features

- 📊 Interactive multi-page dashboard
- 🌡️ Temperature Analysis
- 💧 Humidity & Pressure Analysis
- 🌬️ Wind Speed & Wind Direction Analysis
- 🗺️ Geographical Weather Analysis using Maps
- 📈 Trend Analysis over Time
- 🎯 Dynamic KPI Cards
- 🔥 Hottest & Coldest City Identification
- 🌈 Conditional Formatting & Heat Maps
- 🎛️ Interactive Slicers and Navigation Buttons
